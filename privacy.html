Politique de confidentialite

Derniere mise a jour : 2 mai 2026

Octane ("nous", "l'application") respecte votre vie privee. Cette politique explique quelles donnees nous collectons, pourquoi, et comment nous les protegeons.

1. Donnees collectees

  • Localisation (pendant l'utilisation uniquement) : votre position GPS est utilisee pour trouver les stations a proximite. Elle n'est jamais stockee sur nos serveurs ni partagee avec des tiers. L'application demande uniquement la permission "While Using" (pendant l'utilisation).
  • Adresse email : si vous creez un compte, nous stockons votre email pour l'authentification. Il n'est utilise a aucune autre fin.
  • Preferences vehicule : type de carburant, consommation, rayon de recherche. Stockees localement sur votre appareil et dans votre profil si vous etes connecte.
  • Stations favorites et alertes : stockees dans votre profil utilisateur pour la synchronisation entre appareils.
  • Token de notification push : si vous activez les alertes, un identifiant technique est stocke pour vous envoyer des notifications.

2. Donnees NON collectees

  • Aucun historique de localisation
  • Aucune donnee de contacts, photos, ou fichiers
  • Aucun identifiant publicitaire
  • Aucune donnee revendue a des tiers

3. Base legale (RGPD)

Le traitement de vos donnees repose sur :

  • Consentement : pour la geolocalisation (permission systeme) et les notifications push.
  • Execution du contrat : pour la gestion de votre compte et de votre abonnement Premium.

4. Stockage et securite

Vos donnees sont hebergees sur Supabase (infrastructure AWS, region EU-West-1, Irlande). L'acces aux donnees est protege par des politiques de securite au niveau ligne (Row Level Security). Les communications sont chiffrees en HTTPS/TLS.

5. Duree de conservation

Vos donnees sont conservees tant que votre compte est actif. En cas de suppression de compte, toutes vos donnees personnelles (profil, favoris, alertes) sont supprimees definitivement dans un delai de 30 jours.

6. Vos droits

Conformement au RGPD, vous disposez des droits suivants :

  • Acces : consultez vos donnees dans l'onglet Profil de l'application.
  • Rectification : modifiez vos informations directement dans l'application.
  • Suppression : supprimez votre compte depuis l'onglet Profil. Cette action est irreversible.
  • Export : vous pouvez demander un export de vos donnees (profil, favoris, alertes) au format JSON.
  • Opposition : desactivez la geolocalisation ou les notifications dans les reglages de votre appareil a tout moment.

7. Services tiers

  • Supabase : hebergement base de donnees et authentification (EU-West-1).
  • RevenueCat : gestion des abonnements in-app.
  • Sentry : suivi des erreurs techniques (aucune donnee personnelle transmise).
  • PostHog : analytics d'usage anonymisees (instance UE).
  • Expo : service de notifications push.

8. Sources de donnees carburant

  • France : data.economie.gouv.fr — Licence Ouverte v2.0
  • Espagne : Ministerio para la Transicion Ecologica (MITECO)
  • Luxembourg : donnees publiques gouvernementales
